複数日分のSNSポストをまとめて生成し、スプレッドシートに貼り付けます。

## 引数
`$ARGUMENTS` で以下を指定できます：
- 日数（必須）: 生成するバッチ数（例: 10）
- 開始日（任意）: YYYY-MM-DD 形式（例: 2026-03-11）

例: `/batch 10 2026-03-11` → 2026-03-11から10日分を生成

---

## ⚠️ 絶対に守るルール

### ポスト生成は必ずClaude自身が行うこと
- **Task tool（サブエージェント）を使ってポスト生成をしてはいけない**
- サブエージェントはルールを知らないため、短くて型のない粗悪なポストを生成する
- 全ポストをClaude自身が直接生成すること

### 生成前に必ず読み込むファイル
- `skills/prompt.md` ← 生成ルール・出力フォーマット・日本語ルール
- `skills/knowledge.txt` ← 型01〜型21（投稿構造パターン21種）
- `skills/themes.txt` ← ペルソナ・頻出ワード100選・テーマ120選

### 文字数基準（必須）
- **単ツイート：200〜500字**
- **ツリー：1ツイートあたり200〜500字**
- これを下回るポストはNGとして書き直すこと

### 型の使い方（必須）
- 1ファイル10本の中で同じ型を2回使わない
- 型01〜型21の構造を実際に適用して書く
- 形式A（単ツイート）6〜7本 + 形式B（ツリー）3〜4本

### 頻出ワード（必ず各投稿に2〜3個入れる）
カテゴリ1（★★★超頻出）: エグい / バコバコ売れる / 爆売れ / 〇〇マソ / 0→1達成 / 通知が鳴り止まない / 万バズ / インプ爆増
カテゴリ2（★★★超頻出）: ポチポチ副業 / 自動販売機を作る / 廃人級の覚悟 / 全振り / 作業興奮 / 見切り発車 / 歯磨きレベルで習慣化 / 指を動かせ / 結果が出るまで黙ってやれ
カテゴリ4（比喩・積極活用）: 最強の営業マン / コンビニの棚 / 弾数を増やせ / バットを振り続けろ / 自分のクローンを作る / 宝の山がある
カテゴリ5（リフレーミング・積極活用）: ゲリラ企画 / フロントエンド / 属人性 / シュミマネ / 失敗は資産 / ポンコツエピソード

### 日本語ルール（必須）
- 単ツイートは200〜500字の流れる文体（段落ごとに改行）
- 「〜、は〜」→「〜のは〜」に直す（助詞で文をつなぐ）
- 体言止めは締めの1回まで
- かぎ括弧・強調記号は1投稿に2個まで
- 「することができる」→「できる」に省略

### 出力フォーマット（厳守）
```
[ポスト本文]
（200〜500字のポスト本文）
==========
```
ツリーの場合：
```
[ポスト本文]
■1ツイート目
（200〜500字）
■2ツイート目
（200〜500字）
■3ツイート目
（200〜500字）
==========
```

## 実行手順

1. `skills/prompt.md`、`skills/knowledge.txt`、`skills/themes.txt` を読み込む
2. 既存ファイルの最大連番を確認する
3. **Claude自身が**指定日数分のバッチを連続生成し、`generated/generated_posts_XXXX.txt` に保存する（1ファイル10本）
4. 全ファイルをまとめて変換：
   `python convert_tsv.py generated/generated_posts_A.txt ... --clipboard`
5. `python paste_to_sheet.py` でスプレッドシートに自動貼り付け

途中で確認を挟まず、ノンストップで完了まで進めてください。
